Development Guide

Dev Environment

The main differene in the development environment is that it allows you to bring up an empty server with dependencies installed and the source code mounted as a volume.

make dev

Once you’re in, provision the server.

make

Or run it in two parts like this:

make base
make update

In prod, changes from make base are pre-installed in the image whereas changes from make update (.env-affected configurations) are applied at runtime.
